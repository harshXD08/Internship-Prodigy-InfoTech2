# AI Image Generation using Stable Diffusion

## Overview
This project demonstrates text-to-image generation using Stable Diffusion.

## Features
- Generate high-quality AI images from text prompts
- Uses pretrained diffusion models
- Batch image generation
- GPU acceleration using Google Colab

## Example Prompts
- Futuristic AI laboratory
- Smart city with flying cars
- Humanoid robot coding

## Technologies Used
- Python
- PyTorch
- HuggingFace Diffusers
- Stable Diffusion

## Output
Generated images are stored in the generated_images folder.
